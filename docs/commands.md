# Команды бота

## В любом топике группы
| Команда | Описание |
|---------|----------|
| /cabinet | Открыть кабинет администратора (только CRM-админ) |
| /setup | Первичная настройка CRM: создание/проверка топиков и меню |
| /managers | Показать список менеджеров и ролей (только CRM-админ) |
| /add_manager | Добавить менеджера ответом на сообщение пользователя (только CRM-админ) |
| /remove_manager | Удалить менеджера ответом на сообщение пользователя (только CRM-админ) |
| /make_admin | Назначить CRM-администратора ответом на сообщение (только владелец группы) |

## Только в топике Менеджеры
| Команда | Описание |
|---------|----------|
| /panel | Восстановить/обновить закреплённый пульт управления командой |

## Только в топике Лиды (и других топиках заявок)
| Команда | Описание |
|---------|----------|
| *(inline-кнопки карточек)* | Перевод по этапам, отклонение, заметки, напоминания, клонирование |
| *(ForceReply-ввод)* | Ввод суммы, причины отказа, заметки и даты напоминания по конкретной заявке |

## В личных сообщениях (Master Bot)
| Команда | Описание |
|---------|----------|
| /start | Список аккаунтов, создание нового CRM, оплата и доступы |

## В личных сообщениях (Master Bot, только админ сервиса)
| Команда | Описание |
|---------|----------|
| /clients | Панель клиентов: карточки, блокировка, продление, сообщения |
| /stats | Сводная статистика по клиентам и подпискам |

## Команды настройки (первый запуск)
| Команда | Описание |
|---------|----------|
| /start (Master Bot) | Регистрация компании и создание CRM-аккаунта |
| /setup (CRM Bot в группе) | Привязка группы и развёртывание рабочих топиков CRM |
